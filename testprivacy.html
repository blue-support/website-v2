2. Welche Daten können verarbeitet werden?

Je nach aktivierten Features kann Blue ⚡ technische Discord-Daten verarbeiten, zum Beispiel Server-ID, User-ID, Rollen-ID, Kanal-ID, Command-Nutzung, Moderationsaktionen, Ticket-Inhalte oder Konfigurationen.

3. Zweck der Verarbeitung

Daten werden genutzt, um Bot-Funktionen bereitzustellen: Moderation, Tickets, Logs, Automationen, Leveling, Statistiken und Server-Management.

4. Speicherdauer

Daten werden nur so lange gespeichert, wie sie für die jeweilige Funktion benötigt werden oder bis ein berechtigter Löschwunsch umgesetzt wird.

5. Weitergabe

Eine Weitergabe an Dritte erfolgt nicht, außer sie ist technisch notwendig, gesetzlich vorgeschrieben oder zur Bereitstellung des Dienstes erforderlich.

6. Rechte der Nutzer

Nutzer können Auskunft, Berichtigung oder Löschung ihrer gespeicherten Daten anfragen. Kontaktiere uns dazu über den Support Discord.
