Setup in Minuten

Von leerem Server zu sauberem Dashboard-System.

Blue nutzt jetzt ein Web-Dashboard für die Einrichtung. Admins loggen sich mit Discord ein, wählen ihren Server und konfigurieren alle Systeme zentral im Browser.

  • Kanäle und Rollen ohne Setup-Commands auswählen.
  • Verify-, Ticket- und Message-Embeds mit Preview bearbeiten.
  • Moderation, Globalchat, Fun, Community und Security modular aktivieren.
  • Settings werden gespeichert und vom Bot zurück zur Website gesynct.

Security

Mehr Kontrolle für große Communities.

🛡️ Temporäre Strafen

🌐 Sprachschutz

Server können eine bevorzugte Sprache setzen und andere Sprachen automatisch blockieren.
